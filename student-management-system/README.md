# 🎓 Student Management System

A desktop-based **Student Management System** developed during my **Software Developer Internship at OutriX**.

## 📌 Overview

This application provides a simple and efficient way to manage student records through a graphical user interface. It demonstrates the implementation of CRUD operations with local database integration using SQLite.

## ✨ Features

* Add Student Records
* Edit Existing Records
* Delete Student Records
* Search Students
* View All Students
* SQLite Database Integration
* User-Friendly Desktop Interface

## 🛠️ Technologies Used

* Tkinter
* SQLite
* Git
* GitHub
